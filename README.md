## ✨ 功能特性

### 刷题功能
- **五种题型**：单选题、多选题、填空题、问答题、判断题
- **五种刷题模式**：顺序刷题、随机刷题、错题集、间隔复习、收藏模式
- **收藏功能**：收藏重点题目，支持收藏模式专项练习
- **背诵模式**：卡片翻转式背诵，适合记忆型知识
- **选项乱序**：防止死记硬背，真正检验知识掌握

### 学习数据
- **实时统计**：正确率、连续学习天数、掌握数、总用时、平均每题用时
- **学习热力图**：本周每日学习活跃度可视化
- **题型正确率分析**：各题型掌握情况一目了然
- **掌握程度分布**：已掌握 / 复习中 / 未学习

### 专注工具
- **专注计时**：15/25/45 分钟倒计时，计时结束自动提醒
- **每日目标**：自定义每日刷题目标，自动追踪进度

### 数据管理
- **本地存储**：所有学习进度自动保存在浏览器中，无需注册
- **导出功能**：支持导出 TXT / CSV / JSON / 笔记 / 错题集
- **进度持久化**：关闭页面后重新打开，进度自动恢复

### 体验优化
- **深色/浅色主题**：自动保存偏好
- **字体无极调节**：A- / A+ 实时调整阅读字号
- **键盘快捷键**：← → 翻页 · 1-4 选选项 · Enter 提交 · R 重置 · Space 翻转背诵
- **移动端适配**：手机、平板均可流畅使用

## 🚀 快速开始

### 环境要求
- 无需安装任何依赖，只需一个现代浏览器（Chrome / Edge / Firefox / Safari 均可）

### 使用方式

**方式一：直接打开使用**
1. 获取 `index.html` 和 `data.js` 两个文件
2. 双击 `index.html` 即可在浏览器中打开使用

**方式二：托管部署**
将文件放在任意 Web 服务器（如 Nginx、Apache、GitHub Pages、Vercel 等）即可在线访问

**方式三：加密题库（如提供）**
若您获得的是加密版题库，目录下会包含：
- `data.js` — 加密的题库数据
- `decrypt.js` — 解密脚本

打开 `index.html` 后，输入授权码即可自动解锁题库。

## 📖 功能介绍

### 刷题模式

| 模式 | 说明 |
|------|------|
| **顺序** | 按题目顺序逐题刷题 |
| **随机** | 打乱题目顺序，随机出题 |
| **错题** | 仅练习答错的题目，针对性巩固 |
| **复习** | 基于间隔重复算法，推送待复习题目 |
| **收藏** | 仅练习收藏的题目 |

### 答题流程

1. **选择题**：点击选项 → 点击「提交答案」→ 查看正误与解析
2. **判断题**：点击选项 → 点击「提交答案」→ 查看正误与解析
3. **填空题**：输入答案 → 点击「提交答案」或按 Enter → 查看正误与解析
4. **问答题**：点击「显示参考答案」→ 查看答案与解析

### 间隔重复算法

- 答对题目：复习间隔逐步延长（1天 → 3天 → 7天 → 最长30天）
- 答错题目：复习间隔重置为1天
- 连续正确 3 次且间隔 ≥ 7 天：标记为「已掌握」
